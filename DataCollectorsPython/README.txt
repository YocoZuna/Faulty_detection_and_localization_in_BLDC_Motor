Those scripts were created for collecting data from acqiusition system in order to generate training data

- Current_Monitor.py gathers current data and creates RGB images 
- IMU_Monitor.py  gathers data from IMU and creates STFT images 
- RawDataCollector.py gathers raw data from both current and IMU and saves as .txt files
